## Fatal Flaws

### F1. Dimensional error in floor vibration model (Eq. 6; code sub_bass_coupling.py line 602)

The structure-borne pathway -- one of the paper's two quantitative pillars -- contains a dimensional error. The code computes:

    v_floor = np.sqrt(P_floor / (m_floor_total * omega ** 2))

Dimensional analysis:

- P_floor has units [W] = [kg m^2 s^-3]
- m_floor_total * omega^2 has units [kg s^-2]
- P / (m omega^2) has units [m^2 s^-1]
- sqrt(m^2 s^-1) = m s^(-1/2)  <-- NOT velocity [m/s]

The correct steady-state power balance for a damped vibrating floor is:

    P_absorbed = c_damping * v^2 = 2 * zeta_floor * omega * m * v^2

giving v = sqrt(P / (2 zeta_floor omega m)), which IS dimensionally correct [m/s].

The current formula is equivalent to assuming zeta_floor = omega/2 ~ 126 at 40 Hz -- a damping ratio 1000x larger than any physical floor. For a realistic concrete floor (zeta ~ 0.05), the correction factor is sqrt(omega/(2 zeta)) ~ 50x, meaning the floor pathway displacement is underestimated by ~50x. Even for heavily damped floors (zeta ~ 0.5), the underestimate is ~16x.

**Impact:** The paper's structure-borne pathway result changes from 14% of threshold to well above 100% of threshold. This would strengthen, not weaken, the paper's thesis (structure-borne dominates), but the quantitative 40x ratio and 14% of threshold claims in the abstract, results (Sec. 3.4), and conclusions are all incorrect. The qualitative conclusion survives; the quantitative pathway budget does not.

The corresponding Eq. 6 in the paper (methods.tex, lines 186-188) contains the same error:

    v_floor = sqrt(eta_f I A_f / (m_f omega^2))

This must be corrected to include a damping term.

## Major Issues

### M1. Table 3 displacement values are inconsistent with the SPL column

Table 3 (results.tex, lines 110-130) purports to show tissue displacement at the EDM spectrum SPLs (100, 112, 115, 114, 110, 106 dB across 20-80 Hz). However, the displacement column shows nearly constant values (~1.94-2.0e-3 um), which I verified are the values computed at CONSTANT 115 dB, not at the varying SPLs listed in the same table.

Independent verification:

| f (Hz) | Table SPL | Table xi (um) | Code xi at table SPL | Code xi at 115 dB |
|--------|-----------|---------------|----------------------|-------------------|
| 20     | 100 dB    | 2.0e-3        | 3.57e-4 (WRONG)      | 2.01e-3 (match)   |
| 40     | 115 dB    | 1.95e-3       | 1.95e-3 (correct)    | 1.95e-3 (match)   |
| 80     | 106 dB    | 1.94e-3       | 6.87e-4 (WRONG)      | 1.94e-3 (match)   |

Only the 40 Hz row (where EDM SPL = 115 dB) is correct. The displacement column was evidently generated with constant SPL, then pasted alongside the variable-SPL column. The ratio column is self-consistent with the (wrong) displacement values but not with the (correct) SPL values.

The paper's main claim (0.35% at 40 Hz / 115 dB) is unaffected because that row is correct, but the table must be regenerated.

### M2. Proportionality constants dropped from coupling formula

Equation 3 (methods.tex, line 57-59) states:

    p_eff(n) = p_inc * (ka)^n

The Junger and Feit derivation for the effective modal pressure on a sphere includes mode-dependent numerical prefactors involving spherical Bessel functions and their derivatives, not just the bare asymptotic order (ka)^n. Since the paper ultimately uses the energy-consistent (reciprocity) approach -- which bypasses this formula entirely -- the error does not propagate to the main displacement results. However, presenting Eq. 3 without the coefficient and then not using it creates confusion. Either provide the full expression or explicitly state it is schematic.

### M3. ISO 2631 perception thresholds are not directly from the standard

The paper cites ISO 2631-1:1997 as the source for perception threshold acceleration values (code lines 252-268). ISO 2631-1 provides frequency-weighted acceleration evaluation methods and exposure guidance, but does NOT directly tabulate perception thresholds in the form used here. The source must be correctly identified. If the values are derived from the Wk weighting curve with a chosen base level, state the derivation explicitly. If from published perception threshold experiments (e.g. Parsons and Griffin 1988), cite those.

### M4. No uncertainty quantification for key claims

The paper's headline numbers (0.35%, ~40x, 14%) are presented without uncertainty bounds, despite depending on parameters with substantial uncertainty:

- Wall Young's modulus E: literature range 0.05-0.5 MPa
- Floor coupling efficiency eta_f: assumed 0.01 with no justification; range 0.001-0.1
- Body transmissibility: 1.8, with literature range 1.2-2.5
- Damping ratio: 0.125 (Q = 4)

Sensitivity check shows the airborne ratio is fortuitously insensitive to E (0.35-0.36% across 0.05-0.5 MPa), but this should be demonstrated in the paper. The floor pathway, even after fixing F1, will be highly sensitive to eta_f and zeta_floor.

### M5. "97% of perceptible energy" is misleading overclaiming

Discussion (line 29) and Conclusion (line 51) state "the structure-borne pathway carries ~97% of the perceptible energy." Neither pathway produces perceptible displacement -- both are sub-threshold. Stating "97% of the perceptible energy" implies the total IS perceptible and 97% comes from one source.

Reword to: "the structure-borne pathway accounts for ~97% of the combined (sub-threshold) displacement budget."

### M6. Frequency-dependent tissue properties ignored without justification

The model uses E = 0.1 MPa and eta = 0.25 at all frequencies (values calibrated at ~4 Hz). Soft tissue exhibits significant frequency-dependent stiffening: at 40-80 Hz, E may increase by 2-5x due to viscoelastic dispersion (e.g. Catheline et al. 2004, JASA 116:3734). This would shift the resonance upward and change the off-resonance penalty.

Either incorporate a simple viscoelastic model or present sensitivity analysis showing robustness to E increasing by 2-5x at sub-bass frequencies.

## Minor Issues

### m1. LaTeX formatting error: mismatched dollar delimiters

The construction `\$+\SI{6}{}\$--\SI{10}{\decibel}\$` appears in at least three locations (abstract line 114, discussion line 110, conclusion line 33). The dollar signs are mismatched and will likely cause compilation errors.

### m2. impedance_transmission computed but never used

The airborne_coupling_ratio() function computes an impedance transmission coefficient T ~ 0.001 that is never used in the actual displacement calculation. Either remove it or document that it is for comparison only.

### m3. Near-field model assumes monopole source

Equation 7 (methods.tex line 207) is for a point monopole. Real subwoofer arrays have complex near-field patterns. Note this simplification explicitly.

### m4. SDOF validity at r ~ 20

At 80 Hz (r ~ 20.3), continuum wave effects in tissue may contribute more than the off-resonance tail of the n = 2 mode. Acknowledge this limitation.

### m5. Gunderson et al. (2016) is a conference abstract

The EDM concert SPL spectrum is based partly on Gunderson et al. (2016), listed as an ASA meeting abstract, not a peer-reviewed paper. The specific third-octave band levels should be traceable to a full-length publication.

### m6. Multi-modal increase overstated as less than 10%

Section 3.7 states higher modes increase displacement by less than 10%. Code verification shows the actual increase is less than 0.2% across 20-80 Hz. Report the actual value.

### m7. Highlight line 71 says ka ~ 0.2 at 80 Hz; actual value is 0.23

Minor inconsistency with Table 1 and code.

## Positive Comments

1. **The pathway partition framing is excellent.** Converting a negative result into a quantitative airborne vs. structure-borne comparison significantly increases the paper's contribution.

2. **The energy-consistent displacement method is correctly implemented and verified.** On-resonance displacement at 120 dB (0.0137 um) matches the Paper 1 canonical value. The pressure-to-energy ratio is exactly sqrt(180) = 13.42 as expected. The energy budget closes to machine precision.

3. **The off-resonance scaling (Eq. 9) is analytically clean.** Verified to 0.0002% agreement with independent hand calculation.

4. **The pew bending resonance analysis is creative and correct.** Euler-Bernoulli calculation verified against manual computation.

5. **All 259 tests pass** (53 for Paper 6, 206 for broader suite). Code is well-structured.

6. **The limitations section is commendably honest**, acknowledging seven specific simplifications.

7. **The airborne pathway result (0.35% of threshold at 40 Hz / 115 dB) is robust.** It is insensitive to reasonable parameter variations and correctly computed.

---

## Summary

The paper poses a good question and arrives at a physically reasonable answer: airborne sub-bass alone is insufficient for visceral perception, and structure-borne vibration dominates. The airborne pathway analysis is rigorous, energy-consistent, and well-verified.

However, the structure-borne pathway has a dimensional error (F1) that underestimates floor displacement by 1-2 orders of magnitude. Correcting this would push the structure-borne pathway above threshold, which paradoxically strengthens the thesis but invalidates specific quantitative claims. Table 3 has a data-generation error (M1).

**Recommended path to publication:**
1. Fix the floor model dimensional error (F1) -- changes numbers, not the story
2. Regenerate Table 3 with correct per-frequency SPL values (M1)
3. Source the perception thresholds correctly (M3)
4. Add minimal uncertainty bounds (M4)
5. Fix the overclaiming wording (M5)
6. Address remaining major and minor issues

The paper is publishable after these revisions. The core physics is sound and the question is worth answering quantitatively.
