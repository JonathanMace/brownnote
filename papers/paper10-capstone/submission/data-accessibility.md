# Data Accessibility Statement

**Paper**: Geometry Mediates Excitation, Resonance, and Identifiability in Fluid-Filled Viscoelastic Shells  
**Target**: Proceedings of the Royal Society A

---

## Code Availability

All source code used to produce the computational results, figures, and parametric analyses reported in this paper is publicly available at:

> **Repository**: [https://github.com/JonathanMace/brownnote](https://github.com/JonathanMace/brownnote)

The repository includes:

- Analytical model implementations in Python (`src/analytical/`)
- Scripts for reproducing all figures and parametric sweeps (`scripts/`)
- A complete test suite for regression validation (`tests/`)
- Jupyter notebooks for exploratory analysis (`notebooks/`)

## Reproducibility

All computational results presented in this paper can be reproduced by executing the analysis scripts provided in the repository. The canonical parameter set used throughout the study is defined in a single configuration module and documented in the repository README.

## Experimental Data

This study is purely analytical and computational. No experimental data were collected or used. All results derive from the mathematical models described in the paper and implemented in the code repository.

## Additional Materials

Companion papers in the Browntone programme (Papers 1–9), referenced throughout the manuscript, use the same codebase and canonical parameter set. Their individual results are reproducible from the same repository.
