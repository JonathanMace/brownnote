\section{Results}\label{sec:results}

\subsection{Sphere: rank deficiency}

For the canonical abdomen parameters and modes $n = 2, 3, 4$, the
equivalent-sphere Jacobian has condition number
$\kappa \approx 10^{10}$, confirming near-perfect rank deficiency.  The
singular value decomposition reveals that the smallest singular value is
$\sim 10^{10}$ times smaller than the largest: the three parameters $(a, c,
E)$ are effectively collapsed onto a single degree of freedom.

Physically, this occurs because the equivalent-sphere model maps $(a, c)$
to a single radius $R_\text{eq} = (a^2 c)^{1/3}$, so changes in~$a$ and~$c$
that preserve~$R_\text{eq}$ are invisible to the spectrum.

\subsection{Oblate spheroid: full rank}

The Ritz oblate model yields $\kappa \approx 83.5$ --- a well-conditioned
problem.  All three singular values are of comparable magnitude, meaning
each parameter direction is independently observable from the frequency data.

\subsection{Round-trip inversion}

Starting from a $10\%$-perturbed initial guess, the Newton--Raphson solver
recovers all three parameters to sub-percent accuracy in fewer than 20
iterations, with final residuals below \SI{0.001}{Hz}.

\subsection{Noise sensitivity}

The Cram\'er--Rao bounds scale linearly with the noise level, as expected.
Halving the measurement noise halves the minimum achievable parameter
uncertainty.

\subsection{Watermelon validation}

As an independent validation, we repeat the analysis with watermelon rind
parameters ($a = \SI{0.158}{m}$, $c = \SI{0.123}{m}$,
$E = \SI{50}{MPa}$).  The oblate Ritz model again yields a well-conditioned
Jacobian, and round-trip inversion succeeds to sub-percent accuracy.
