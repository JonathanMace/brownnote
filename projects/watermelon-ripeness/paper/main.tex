\begin{abstract}
The `thump test''---tapping a watermelon and listening for
ripeness---is practised worldwide but lacks a quantitative physical
model.  We model the watermelon as a fluid-filled viscoelastic shell
using an equivalent-sphere approximation: the oblate geometry is
captured through the equivalent-volume radius
$R_{\mathrm{eq}} = (a^{2}c)^{1/3}$, and the eigenfrequency is
derived from classical thin-shell theory applied to a sphere of that
radius.  Sobol sensitivity analysis
($N_{\mathrm{base}}(D{+}2) = 5{,}632$ evaluations) reveals that the
rind elastic modulus $E_{\mathrm{rind}}$ dominates the
frequency response (total-order index $S_T = 0.54 \pm 0.05$).  Because $f_2^2$
is exactly linear in $E_{\mathrm{rind}}$ when turgor pressure is
negligible, the forward model admits a closed-form inversion: a single
tap-tone measurement yields the rind modulus, which correlates with
ripeness through pectin degradation.  The condition number of this
inversion is unity with respect to $f^2$ and approximately~2 with
respect to measured frequency~$f$.  We construct a dimensionless
geometric invariant $\Pi_{\mathrm{ripe}}$ that collapses
predictions across cultivar geometries onto a single calibration
constant; because $E_{\mathrm{rind}}$
cancels by construction, $\Pi_{\mathrm{ripe}}$ characterises the
shell--fluid geometry rather than ripeness \emph{per se}.  Predictions
are benchmarked against published resonance frequency ranges for
watermelons.  The framework
generalises directly from prior work on biological cavities,
demonstrating that the same equivalent-sphere shell theory governs both
biomedical vibroacoustics and agricultural non-destructive evaluation.
\end{abstract}
